# AstNet-upd

AstNet-upd — утилита для автоматического обновления AstNet.

## Установка
1. Убедитесь, что AstNet установлен (см. [основной репозиторий](https://github.com/AstreelX/AstNet)).
2. Скачайте последнюю версию бинарного файла из [релизов](https://github.com/AstreelX/AstNet-upd/releases).
3. Распакуйте и настройте интеграцию с AstNet для seamless обновлений.

## Лицензия и доступ
Вспомогательное закрытое ПО. Требует действующей лицензии на AstNet. Исходный код не открыт.

Для получения лицензии: contact@astreel.net

⚠️ **Только для лицензированных пользователей AstNet.**
