# AstNet

AstNet — инструмент для безопасного и быстрого доступа в интернет.

## Установка
1. Скачайте последнюю версию бинарного файла из [релизов](https://github.com/AstreelX/AstNet/releases/).
2. Распакуйте и запустите исполняемый файл.
3. Для автоматических обновлений интегрируйте AstNet-upd, он необходим для работы приложения (см. [репозиторий обновлений](https://github.com/AstreelX/AstNet-upd)).
